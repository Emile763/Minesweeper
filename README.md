# How To Compile
At the root of the repository:
```console
mkdir build
cd build
cmake ..
cmake --build .
# To launch
./RenderApp
```

# How To Use
<kbd>Left Click</kbd> : Uncover a cell\
<kbd>Right Click</kbd> : Place/Remove a flag
